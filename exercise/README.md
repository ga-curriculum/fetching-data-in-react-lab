# ![Fetching Data in React Lab - Exercise](./assets/hero.png)

In this lab, you will be retrieving data from the [Star Wars API (SWAPI)](https://swapi.dev/) and inserting it into the DOM.

Take some time to explore [the documentation](https://swapi.dev/documentation) to gain some insight into how you can use it.  Figuring out documentation is a critical skill for a developer, so devote 10-15 mins just to looking through everything.  You're not going to understand it all, but you should learn a bit in the process.  There is also a very nice [home page](https://swapi.dev/) that allows you test out calls that you'd like to make.  Try making some calls using this tool as well.

Once you have done that, pick one of the "Resources" listed on the left of the [documentation page](https://swapi.dev/documentation) (e.g. People, Starships, Species, Planets, etc...) and build a user interface so that the user can enter a number into an input, submit the form, and retrieve specific information about the chosen "resource."  Reference the lesson notes to figure out how to do this.
